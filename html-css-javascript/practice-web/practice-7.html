Proportional Border Radius

You can use this formula when there's padding between the parent and child elements to keep the proportions correct.

Formulas :

child-radius = calc(parent-radius - padding)

parent-radius = calc(child-radius + padding)

CSS Code Snippet : Using Prism.js

     
            
    .parent {
    width: 16.5rem;
    padding: 30px;
    border-radius: 40px;
    background-color: #f0f0f0;
    margin: auto;
    }

    .child {
    border-radius: calc((40 - 30) * 1px);
    background-color: #00f;
    color: #fff;
    border: none;
    }

Example :

The child's border-radius is reduced by the padding amount to maintain the proportional curvature inside the parent. If you increase padding, the child's border-radius decreases accordingly.This ensures smooth, consistent rounding between the two elements.
